# 🎴 Animated Cards Experience

An interactive 3D animated card gallery built with **React Three Fiber**, **Three.js**, **GSAP**, and **Leva**.
The project showcases smooth 3D animations, dynamic image textures, and an immersive user experience powered by WebGL.

## 🚀 Live Demo

🌐 **Live Website:** https://fan-animation-wine.vercel.app/

## ✨ Features

* 🎨 Interactive 3D card animation
* ⚡ Smooth GPU-accelerated rendering using React Three Fiber
* 🌍 Powered by Three.js
* 🖼️ Dynamic image textures
* 🎛️ Real-time controls with Leva
* 🎥 Responsive 3D Canvas
* ✨ Smooth animations
* 📱 Responsive design for multiple screen sizes
* ⚡ Fast development with Vite
